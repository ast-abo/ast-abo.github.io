Step 1: Starting Out

Head over to github and create a repository in your name, if you do not have a github account, create one!

Properly name your repository as "Username/Username.github.io"

After creating a repository create your first codespace

Now you should be here!

Step 2: Setting up Your Codespace

First create two files, name one html.index and the other style.css

Go to exstensions, search up "Live Server", and install it

Now you can see a live version of your website if you right click "index.html and click "Open with LiveServer"

go back to the html.index file and create the "structure" of this file
