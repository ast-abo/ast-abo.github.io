Step 2: Setting up Your Codespace

First create two files, name one html.index and the other style.css

Second, to exstensions, search up "Live Server", and install it

Now you can see a live version of your website if you right click index.html and click "Open with LiveServer"

Lastly, you want to change your theme go to exstentions and search theme untill you find the theme exstension.

go back to the html.index file and create the "structure" of this file

The "body" and "head" are known as tags, I will refer to them as containers however
In the head tag connect the style file to your html file and set your page title. The body tag will hold most of your text.
